## Key Decision Points

### When to Show Signup Modal?
```javascript
// In reflection.tsx
const handleSaveToJournal = async () => {
  const userId = await getUserId();

  if (!userId) {
    setShowSignupModal(true);  // ‚Üê Show modal
    return;
  }

  // Save to Supabase
  await saveJournalEntry({...});
};
```

### When to Show Signup Gate?
```javascript
// In index.tsx (home screen)
useFocusEffect(() => {
  const userId = await getUserId();
  const flows = await getFlowsCompleted();

  if (!userId && flows >= 1) {
    setShowSignupGate(true);  // ‚Üê Block access
  }
});
```

### When to Increment Flow Count?
```javascript
// In reflection.tsx
const handleStartOver = async () => {
  const userId = await getUserId();

  if (!userId) {
    await incrementFlowsCompleted();  // ‚Üê Track usage
  }

  router.replace('/');
};
```
